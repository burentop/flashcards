Learn Javascript and jQuery Terms

Click on a term to see its definition; click the definition to hide it and go back to the term.

JavaScript

A scripting programming language, most commonly used to modify the behavior of a web page.

Operators

Constructs which are similar to functions; allowing terms to be combined in some manner.

Variables

Used to store information in a computer program.

Variable naming conventions

Rules, and best practices, used when creating variable names. Some words are reserved by the system.

Functions

A named sequence of program instructions that performs a specific task.

Methods

Similar to a function, but is associated with a particular object (in OOP).

Arguments

Information passed when calling a function.

Parameters

A name used in a function declaration to hold information.

Return

A value that is the result of the completion of a function.

Chaining methods

The practice of calling a method on the results of another method.

Strings

A sequence of characters with no mathematical consequence.

Boolean

A data type that can hold the values 'true' or 'false'.

Undefined

The result of a function with no return statement, or an unassigned variable.

NaN

Means 'Not a Number', but it is in fact a numerical 'type' in JavaScript.

Escape

Used to indicate that a special character (normally reserved for some special calculation) to be used as a literal. Precede the character with '\'.

Alert

A method that creates a pop-up box with information.

Comments

Portion of a computer program that is ignored by the computer. Used as instructions for humans.

jQuery

A JavaScript library used to simplify the client-side scripting of HTML.

Attributes

The property of an object (OOP) - (e.g. whether a border is solid or dashed).
